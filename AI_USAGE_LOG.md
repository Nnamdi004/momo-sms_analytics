# AI Usage Log

## Tools Used
- ChatGPT (OpenAI)

## Purpose of AI Usage
AI tools were used strictly for:
- Understanding JSON serialization concepts
- Grammar and wording improvements in documentation
- Clarification of technical terms related to APIs and databases
- Example JSON structures for data representation

## What AI Was NOT Used For
AI was not used to:
- Design the ERD
- Generate SQL schemas
- Define database relationships
- Create business logic
- Write technical design rationale

## Human Contribution
All database design decisions, ERD modeling, SQL implementation,
and system architecture were created and reviewed by the team members.

AI assistance was used only as a learning and support tool.
